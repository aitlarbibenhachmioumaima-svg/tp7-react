# TP 7 : Comprendre JSX, la composition avancée et les tests React

## Objectif

Ce TP a pour but de te faire :

- Comprendre JSX et sa transformation en JavaScript pur.
- Créer des composants réutilisables avec les **Higher-Order Components (HOC)** et **Render Props**.
- Écrire des tests unitaires et d’intégration avec **Jest** et **React Testing Library**.
